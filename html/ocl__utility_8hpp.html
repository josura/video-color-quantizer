Detailed Description

A modern C++ utility header for managing common OpenCL boilerplate code.

This header provides helper functions for selecting platforms and devices, creating OpenCL contexts and queues, compiling programs, and profiling events. The goal is to minimize redundant code when initializing and managing OpenCL.

Adapted and extended from a C version for use with C++.
