Video Quantization with OpenCL and FFmpeg

Introduction

This project performs uniform color quantization on each frame of a video using GPU acceleration (OpenCL). It utilizes FFmpeg to decode input videos into raw frames, applies quantization kernels on the GPU, and encodes the processed frames back into a final video output.
